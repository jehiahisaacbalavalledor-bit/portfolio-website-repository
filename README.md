# portfolio-website-repository
This is the repository for my portfolio website project

# PARTS OF THE WEBSITE

## The Hero Page
- Also known as the "Home" page, this is the main page that you will first see when you click the link of my website.

- You can also see the basic info and what I do in the hero page.

## About Page
- This is where you can see my skills, hobbies, experiences, etc.

- The profeciency in commands and programming languages is also displayed in this section.

## Projects Page
- This is where I display the projects that I have done with my experience in coding

## Contact Page
- You can see the buttons that lead to my profiles in different platforms and also my email link and contact number.

## Interactable Sidebar
- This is where other menus will be displayed after clicking the hamburger icon at the top of the header in the navigation bar.

### Parts of the Interactable Sidebar:
+ Download my Resume - You can download my resume by clicking this menu
+ Schedule a Meeting - You can interact with a calendar widget that set the date to schedule a meeting
+ Hire Me - This will lead to a page where you see the services that I do
+ Settings - This will change the settings like light and dark mode
